# GlaucoVision 👁 — Multimodal Glaucoma Detection

Automated glaucoma screening system combining **Fundus photography** and **OCT B-scan** imaging using deep learning.

## Architecture

| Component | Detail |
|-----------|--------|
| Fundus model | ResNet50 (ImageNet pretrained) + custom binary head |
| OCT model | ResNet50 (ImageNet pretrained) + custom binary head |
| Fusion | Weighted average — α=0.49 (Fundus) + 0.51 (OCT) |
| Training | StratifiedKFold K=5, BCEWithLogitsLoss + pos_weight, AdamW |
| Augmentation | Albumentations — CLAHE, affine, flip, GaussNoise, CoarseDropout |

## Performance (OOF — 48 pairs)

| Method | AUC | Accuracy | Sensitivity | Specificity |
|--------|-----|----------|-------------|-------------|
| Fundus only | 0.695 | 60.0% | 40.0% | 74.7% |
| OCT only | 0.878 | 72.4% | 60.0% | 82.0% |
| **Weighted Fusion ★** | **0.802** | **75.0%** | **70.0%** | **78.6%** |

## How to Use

1. Upload a **Fundus** (color retinal photo) image
2. Upload an **OCT B-scan** image
3. Click **Analyze Images**
4. Review the fusion result showing probability per model

## Model Weights

> ⚠️ This Space runs in **Demo Mode** until model weights are uploaded.

To enable real inference, upload your trained weights to the `models/` directory:
- `models/fundus_model_best.pth` — from notebook `03_fundus_model.ipynb`
- `models/oct_model_best.pth` — from notebook `04_oct_model.ipynb`

These `.pth` files are generated by Kaggle notebooks 03 and 04 (see project source).

## Dataset

- **Source**: Mendeley Data — *Data on OCT and Fundus Images* (marwanenasri)
- **Size**: 48 valid Fundus+OCT pairs (26 patients)
- **Labels**: Majority vote of 4 ophthalmologists
- **Classes**: 20 Glaucoma / 28 Normal

## ⚕️ Medical Disclaimer

This tool is for **research and screening assistance only**. It does not replace clinical diagnosis by a qualified ophthalmologist. Results should never be used as the sole basis for medical decisions.
